API Basejump: Request Header Parser microservice

User stories:
    1) I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
    2) If it does, it returns both the Unix timestamp and the natural language form of that date.
    3) If it does not contain a date or Unix timestamp, it returns null for those properties.

Go to:

https://fcc-nodejs-playground-jackyef.herokuapp.com/api/timestamp/1450137600
https://fcc-nodejs-playground-jackyef.herokuapp.com/api/timestamp/December%2015%2C%202015

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }

API Basejump: URL-shortener microservice

Go to:

https://fcc-nodejs-playground-jackyef.herokuapp.com/api/shorten-url/https://www.google.com

Example output:

{ "original_url":"https://www.google.com", "short_url":"https://fcc-nodejs-playground-jackyef.herokuapp.com/api/shortened-url/237HAds" }
Then you can go to https://fcc-nodejs-playground-jackyef.herokuapp.com/api/shortened-url/237HAds and be redirected to https://www.google.com
